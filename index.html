JavaScript (JS)

JavaScript helps you move from a static web page created through HTML and CSS to dynamic and interactive style web programming

JQuery

JQuery contains a library of events that allows the web program to move from one DOM (Direct Object Manipulator to another. You can think of this as moving from one "state" to another)

NaN

"NaN" is a dedicated number in HTML that is decidated to non-numbers. It stands for Not a Number"

Alert

The function Alert("insert string") is used to display a message in the browser. It can be used in HTML or as a function in Javascript. Example javascript code:$("Variable").click(function(){alert("this is a warning!")});
